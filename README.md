# Modern Login Page

A beautiful and interactive login page built with Vue 3, Vite, Tailwind CSS, and GSAP.

## Features

- 🎨 Modern UI design with purple/indigo color scheme
- 🎭 Animated geometric characters (circle, square, capsule)
- 🌐 Multi-language support (English / 中文)
- 🔐 Forgot password functionality
- 📝 Create account functionality
- ✨ Smooth animations powered by GSAP
- 📱 Responsive design

## Tech Stack

- Vue 3 (Composition API)
- Vite
- Tailwind CSS
- GSAP (GreenSock Animation Platform)

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build
